# 座位自动预约bot

一个非常简单轻量的抢座小工具，适用于Windows网页、微信小程序、本地应用程序的定时抢座。

> **面向群体**：需要抢自习室、图书馆座位的大学生朋友们。本项目通过 AI 识别屏幕，模拟人工点击，实现定时、无人值守的自动化预约。当前支持一考即过座位预约小程序，其他预约平台没测过。可以自行在运行时修改提示词，简单描述流程即可。

---

## 🚀 快速上手

### 第 0 步：准备 Windows Python 环境
1. **安装 Python**：打开 Windows 的 **Microsoft Store** (微软商店)，搜索 "Python"，点击安装最新版本 (如 Python 3.12)。这是最简单的方法，安装时请务必勾选 **"Add Python to PATH"**。
2. **下载代码**：确保你已经下载了本项目的所有文件并解压。
3. **一键启动 (推荐)**：直接双击目录下的 **`装完Python双击我启动程序.bat`**。它会自动为你安装所有依赖并启动程序。

### 第 1 步：获取 AI 密钥并运行
1. **获取 API Key**：前往 [阿里云百炼控制台](https://bailian.console.aliyun.com/)，在“常用功能”中点击 **[API Key]**，创建一个 `sk-` 开头的密钥。

![阿里云百炼主页](docs/assets/bailian_home.png)


2. **启动程序**：双击运行 **`装完Python双击我启动程序.bat`**。
3. **交互式配置**：如果检测到没有配置密钥，程序会**直接弹出输入框**请你输入密钥。输入后程序会自动保存到 `.env` 文件并正常启动，之后无需重复配置。

### 第 2 步：自动化定时设置
1.  **静默运行**：你会发现运行的是 `app.pyw`，它不会像传统的 `.py` 文件那样带着一个黑色的命令行窗口，界面更加干净。
2.  **参数配置**：
    - 在界面上填好 **[1. 抢座开始时间]** (如 06:00) 和 **[2. 预约时段设置]**。
    - 点击 **[💾 保存并设置为默认]**。
3.  **锁定小程序**：
    - 在电脑端微信打开预约小程序。
    - 在本程序点击 **[🎯 手动点击锁定窗口]**，然后立即点击一下微信小程序窗口内部。
4.  **开启模式**：
    - **立即测试**：模拟一次流程，最后一步不提交，用于排查你的提示词写得对不对。
    - **开始定时模式**：到点自动准时开抢，无需人工干预。

## 🎓 进阶：如何适配“你自己学校”的流程

每个学校的预约小程序长得都不一样，自习室、图书馆名字资源也肯定不一样。默认流程和你的学校几乎一定对不上，你需要修改 **AI 的行为规则**。

### 1. 修改方式
在程序界面点击 **[📝 提示词配置]** 标签页：
- 点击保存后，会写入 `user_system_promote.txt`（用户自定义提示词，优先级最高）。
- 若未保存用户提示词，则默认读取项目根目录下的 `default_system_prompt.txt`（建议通过 Git 同步维护）。
- 若 `default_system_prompt.txt` 缺失或损坏，程序会直接报错，请重新拉取 Git 仓库或重新下载完整项目。

### 2. 核心修改逻辑
找到提示词中的 `# 这个微信小程序的完整预约步骤` 部分，把它改成你学校的真实步骤。

**修改示例：**
如果你的学校需要先点“入馆预约”再点“楼层”，你可以这样写：
> 1. 首页中间有个大图标叫「入馆预约」，点击它。
> 2. 进入后看到楼层列表，找到「三楼阅览室」旁边的按钮点击。
> 3. ... (以此类推)

**修改锦囊：**
- **像教小朋友一样**：把每个按钮的**文字内容**、**颜色**、**位置**描述清楚。
- **利用占位符**：保留 `{seat1}`、`{start_time}` 等大括号，AI 会自动把你界面填写的座位号和时间填进去。
- **坐标不要管**：你只需要描述“点哪里”，AI 会自己看截图找坐标。

## 🛠️ 配置说明 (config.py)
如果你需要修改默认的预约时间或座位，可以修改 `config.py`：
- `PREFERRED_SEATS`: 你的心仪座位号，比如 `["A101", "A102"]`。
- `TRIGGER_HOUR`: 每天几点自动开始抢（默认早上 6 点）。
- `AGENT_MAX_STEPS`: Agent 单次任务的最大循环步数（默认 `50`，推荐 `20~60`）。
- `DRY_RUN`: 设为 `False` 时，点击“测试”也会真的提交预约（请谨慎）。
