# 座位自动预约bot

一个非常简单轻量的抢座小工具，适用于Windows网页、微信小程序、本地应用程序的定时抢座。

> **面向群体**：需要抢自习室、图书馆座位的大学生朋友们。本项目通过 AI 识别屏幕，模拟人工点击，实现定时、无人值守的自动化预约。当前支持一考即过座位预约小程序，其他预约平台没测过。可以自行在运行时修改提示词，简单描述流程即可。

---

## 🚀 快速上手

### 第 1 步：获取你的 AI 访问凭证 (API-KEY)
本程序通过阿里云的**通义千问大模型**来识别界面，你需要先申请一个免费的 API Key（新用户有赠送额度，个人使用足够用很久）。
当然，你也可以接自己的大模型，建议用好一点的，本项目默认使用qwen3.5plus。

**详细图文向导：**
1.  **打开控制台**：前往 [阿里云百炼控制台](https://bailian.console.aliyun.com/) (建议在电脑浏览器打开)。
2.  **直接点击 API Key 入口**：进入主界面后，在 **“常用功能”** 区域直接就能看到 **[API Key]** 选项（如下图所示）。

    ![阿里云百炼主界面](docs/assets/bailian_home.png)

3.  **创建密钥**：
    *   点击蓝色的 **[创建 API KEY]** 按钮。
    *   弹窗出来的长字符串（以 `sk-` 开头）就是你的 API-KEY。
    *   **【极其重要】**：点击“复制”并放备忘录存起来。
    *   **不要发给别人！** 这相当于你的充值卡密码。

### 第 2 步：配置程序环境变量
1.  在本项目文件夹根目录下，你会看到一个文件叫 `.env.example`。
2.  把它**复制一份**，改名为 `.env`（注意前面有个点，没有后缀名）。
3.  右键用记事本打开 `.env`，把 `API_KEY=` 后面的内容删掉，换成你刚才复制的 `sk-...` 密钥。
    ```text
    API_KEY="sk-你的真实密钥"
    ```
4.  保存并关闭。

### 第 3 步：运行并锁定窗口
1.  启动程序：双击运行 `main.py`（或者在终端输入 `python main.py`）。
2.  **锁定小程序**：
    *   先在电脑端微信打开你的自习室预约小程序，调整好窗口大小。
    *   回到本程序界面，点击 **[🎯 手动点击锁定窗口]**。
    *   **马上**去微信小程序窗口内随便点一下，程序日志显示“成功锁定窗口（务必确保和你的目标窗口标题一致）”即可。
3.  **测试运行**：点击 **[🛠️ 立即测试]**。程序会模拟一遍流程，但在最后一步提交前会停下，让你检查 AI 点的对不对。

## 🎓 进阶：如何适配“你自己学校”的流程

每个学校的预约小程序长得都不一样，自习室、图书馆名字资源也肯定不一样。默认流程和你的学校几乎一定对不上，你需要修改 **AI 的行为规则**。

### 1. 修改方式
在程序界面点击 **[📝 提示词配置]** 标签页，或者直接修改项目根目录下的 `system_prompt.txt` 文件。

### 2. 核心修改逻辑
找到提示词中的 `# 这个微信小程序的完整预约步骤` 部分，把它改成你学校的真实步骤。

**修改示例：**
如果你的学校需要先点“入馆预约”再点“楼层”，你可以这样写：
> 1. 首页中间有个大图标叫「入馆预约」，点击它。
> 2. 进入后看到楼层列表，找到「三楼阅览室」旁边的按钮点击。
> 3. ... (以此类推)

**修改锦囊：**
- **像教小朋友一样**：把每个按钮的**文字内容**、**颜色**、**位置**描述清楚。
- **利用占位符**：保留 `{seat1}`、`{start_time}` 等大括号，AI 会自动把你界面填写的座位号和时间填进去。
- **坐标不要管**：你只需要描述“点哪里”，AI 会自己看截图找坐标。

## 🛠️ 配置说明 (config.py)
如果你需要修改默认的预约时间或座位，可以修改 `config.py`：
- `PREFERRED_SEATS`: 你的心仪座位号，比如 `["A101", "A102"]`。
- `TRIGGER_HOUR`: 每天几点自动开始抢（默认早上 6 点）。
- `DRY_RUN`: 设为 `False` 时，点击“测试”也会真的提交预约（请谨慎）。
